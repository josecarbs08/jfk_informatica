# JFK Informática

Repositorio de trabajos y recursos del área de **Informática** organizados por semestre. Incluye prácticas en **HTML/CSS/JavaScript** y ejercicios en **Python**, además de materiales complementarios.

> **Estado:** material en construcción y en constante actualización.

## Estructura del repositorio

- **Semestre 1/**
  - **Dibujo/**: práctica web (HTML, CSS y JavaScript) y recursos (imágenes).
- **Semestre 2/**
  - **Python/**: colección de ejercicios en Python (archivos `*.py`) y carpeta de apoyo `librerias/`.
- **Mounstros6B/**
  - Mini proyecto web con `index.html`, `style.css`, `script.js` y sprites/animaciones (`.png`, `.gif`).
- **.vscode/**
  - Configuración del editor para desarrollo (Visual Studio Code).

## Requisitos

Depende del contenido que se quiera ejecutar:

### Para proyectos web (HTML/CSS/JS)
- Un navegador moderno (Chrome, Edge, Firefox).
- (Opcional) Visual Studio Code + extensión **Live Server**.

### Para ejercicios de Python
- **Python 3.x** instalado.

## Uso

### Ejecutar proyectos web

1. Abrir la carpeta del proyecto (por ejemplo `Semestre 1/Dibujo/` o `Mounstros6B/`).
2. Abrir el archivo `index.html` en el navegador.
3. (Opcional) Si usas VS Code, inicia **Live Server** para recarga automática.

### Ejecutar ejercicios de Python

Desde la terminal, ubicándote en la carpeta correspondiente:

```bash
python "Semestre 2/Python/1.- ejercicio1.py"
```

> Nota: algunos archivos pueden estar vacíos o en progreso.

## Convenciones

- Las carpetas están organizadas por **semestre** y **tema**.
- Los ejercicios de Python siguen el patrón: `N.- ejercicioN.py`.

## Contribución

Si vas a seguir subiendo tareas o mejoras:

- Mantén la estructura por semestre.
- Añade una breve descripción en el commit (ej. `Agrega ejercicio 11 Python`).
- Si agregas un proyecto nuevo, crea su carpeta con un `index.html` (si es web) o un `main.py` (si es Python).
